# Healthcare Management System

This project is a Healthcare Management System developed using React and Spring Boot. 
It helps hospitals manage patients, doctors, appointments, and medical records in an organized way.

## Project Overview
The system provides a unified platform for patients, doctors, and receptionists to manage healthcare services efficiently.

## Features
- Patient registration and login
- Doctor management
- Appointment booking
- Medical records management
- Role-based access for users

## Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS
- Spring Boot
- REST APIs
